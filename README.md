# TradeHub

Simple marketplace app with a React frontend and a Node.js/Express backend.

## What is inside
- `mad/` - React app (Vite)
- `mad market backend/` - Backend API (Node.js + Express)

## Features
- Browse items and search with filters
- Buyer and seller flows
- Chat between buyer and seller
- Seller dashboard for listings

## Setup (Frontend)
1) Open a terminal in `mad/`
2) Install packages:
   - `npm install`
3) Run the app:
   - `npm run dev`

## Setup (Backend)
1) Open a terminal in `mad market backend/`
2) Install packages:
   - `npm install`
3) Start the server:
   - `npm run dev`

## Notes
- Set your backend URL in the frontend if needed.
- For production, run `npm run build` in the frontend.
